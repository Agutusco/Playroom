Información general

Playroom es un sitio web creado por Agustín Lombardi de 17 años. Con este proyecto, se busca que la gente pueda pasar un rato jugando, que sea su espacio para despejarse y poder pensar. En Playrrom incluimos a todos.

Es por eso que el sitio web ofrece distintos juegos como también distinta música. Es un proyecto en comunidad que busca integrar a todos y que puedan aportar ideas y mejoras para el sitio web.

Actualmente el sitio solo se encuentra en GitHub bajo el usuario de Agutusco. Día a día y ganando más experiencia, se busca poder mejorar el sitio web para algún día subirla oficialmente a la web para que cualquier persona pueda entrar y pasar un rato en esta zona de entretenimiento.
